# `@datasqrl/acorn` usage example with `openai`.

This example contains Rick and morty API powered AI agent implementation.

It creates tools from GraphQL API. And performs a request to the chatbot with
these tools included.

_Please take a note `openai` package must be installed in your project to
use `@datasqrl/acorn/openai`_

## To run an example script

1. Install dependencies

```sh
npm ci
```

2. Run example script

```sh
export OPENAI_API_KEY=sk-...
npm run start
```
